# 💼 Amaan Siddiqui's Developer Portfolio

A modern, responsive developer portfolio built using **React.js** via Create React App, styled with **Bootstrap**, and enhanced with animation and interactivity libraries. It showcases personal projects, skills, and contact information in a professional, single-page format.

---

## 📌 Features

- ⚛️ Built with **React.js** (Create React App)
- 🎨 Styled using **Bootstrap 5**
- ✨ Smooth scroll-based animations using **AOS (Animate On Scroll)**
- 🔁 Dynamic loading screen with rotating quotes
- 🧠 Organized project showcase with background images and hover effects
- 🌐 Social and contact links (GitHub, LinkedIn, Email, Phone)
- 🖼️ Responsive design for all screen sizes
- 📄 Resume download option
- 🧩 Icons powered by **Lucide React**

---

## 🛠️ Tech Stack

- **Frontend**: React.js, Bootstrap 5
- **Animations**: AOS (Animate on Scroll)
- **Icons**: Lucide React
- **Package Manager**: npm
- **Deployment**: Suitable for GitHub Pages, Vercel, Netlify, etc.
